# ⚖️ Legal Translator (RAG)

A Local RAG application built on **M4 Mac** using **DeepSeek-R1** and **Ollama** to analyze and translate complex Terms of Service documents.

### 🚀 Tech Stack
* **LLM:** DeepSeek-R1 (7B) via Ollama
* **Framework:** LangChain 1.x (2026)
* **Vector DB:** ChromaDB
* **UI:** Streamlit
* **Embeddings:** Nomic-Embed-Text

### 🛠️ Setup
1. Clone the repo: `git clone <repo-link>`
2. Install requirements: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
